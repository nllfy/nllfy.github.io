Introduction

The csh-framework is designed to be a cyber security and networking tool

Installation

  1. Clone the repository: 
    git clone https://github.com/nllfy/csh-framework.git
  2. Navigate into the directory: 
    cd csh-framework
  3. Install dependencies: 
    install dotnet 9 runtime
  4. install the project: 
    cd bin/Debug/net9.0/
    sudo cp csh-framework /bin

Usage

csh-framework
